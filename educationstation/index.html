What is HTML? What can I create with it?


We use HyperTextMarkupLanguage to create the structure of our web, HTML is like the foundation of the house, to design the house, we use CSS, and to make the house a functional home, we use Javascript.

<html> <!--You can copy paste this code to text editor, save it with index.html and run it in your browser.--> <head> <!-- This is a comment, everything written here wont be executed by the compiler. --> <!-- We use <head> to mainly put our metadata, metadata typically define the document title, character set, styles, links, scripts, and etc --> <title>Hello World!</title> </head> <body> <!-- Inside the body tag the to content of the document --> <h1>This is header</h1> <p>This is paragraph </p> <a href="www.google.com">This is anchor <!-- It opens google after you click it--> </a> <img src="banana.jpg" > <!--It doesn't insert banana image because there is no banana.jpg in your pc. Now what? Well now we paint the house with CSS, CSS is short of Cascade Style Sheet, and we use it to design the web, we can also make animation with CSS, cool right? But CSS isnt a programming language, it's a style sheet language, we will get to the programming section after CSS.


There are 3 ways to use CSS in your HTML file

  1. Use style inside the HTML tag
  2. Put style in head tag
  3. With an external file, create other file name style.css or whatever you prefer

The cleanest way to use it to by use external file, but if you use CSS inside your HTML tag, it will overwrite the external file CSS code and style tag in head HTML tag, the priority is same as the number list.

<html> <!--You can copy paste this code to text editor, save it with index.html, and run it in your browser.--> <head> <link rel="stylesheet" type="text/css" href="style.css"> <!--External file of css--> <title>Hello World!</title> <style> body{ /* This is how you comment in CSS background-color : white; */ background-color : black; font-size: 10px; margin-right: auto; position: fixed; display:block; } .answer{ /* We use . before the class name */ font-size: 60px; margin-top: 40%; } #color{ /* We use # before the id name */ color:aqua } </style> </head> <body style="background-color:blue"> <!--Check the background color of your web--> <ol> <li>White</li> <li>Black</li> <li>Blue </li> </ol> <p class="answer">The correct answer is 3</p> <p id="color">which is blue.</p> </body> </html>

Object

In real life, a car is an object. A car has properties like weight and color, and methods like start and stop. All cars have the same properties, but the property values differ from car to car. All cars have the same methods, but the methods are performed at different times.

var human = { "name" : "Edward", "age" : 17, "hobby" : "playing games"; } //Accessing Object Properties with Dot notation console.log(human.name) // expected output : "Edward" console.log(human.age) // expected output : 17 console.log(human.hobby) // expected output : "playing games" //Accessing Object Properties with Bracket notation console.log(human["name"]) // expected output : "Edward" console.log(human["age"]) // expected output : 17 console.log(human["hobby"]) // expected output : "playing games"

That is for the data structure, we use variables, arrays, and objects to store our data, and we can manipulate the data the way we want it.

You can manage the condition inside if else statement by using || which is OR, or && which is AND

OR returns true when one of the condition is met, and AND returns true when both condition met

In this case when you iterate from i to number, 1, 2 , 3 ,... number, you want the codes to execute when it's divided by 3 or divided by 5, so we use ||

We cant use && because the && only makes your code condition works when i is both divisible by 3 and 5, and we want the code to run when i is divided by either 3 or 5.

function findGreater(a, b) { return a > b ? "a is greater" : "b is greater"; }

Tips to create a project.



HTML, CSS, Javascript are tools that we use to create something, everytime you want to create something, start with why, since it will become your motivation to not stressed out and have fun programming.

Done thinking? Now imagine you want to make a game, how will the system work? How do you want the system to work? And what needs to be done to make everything work? You write everything in a docs or a note.

Great, now you know how the game works. Next step is to break everything down to little part, for example the characters, the place where the game is played, the scoring system, the skills of the characters, how the players will send an input, and every single thing.

Things now starting to get clearer. Characters are variables, scoring system and skills are function, inputs are part of HTML.

You also want to think logically on how it works, since you don't want to get stuck a year trying to make something that isnt well planned.
