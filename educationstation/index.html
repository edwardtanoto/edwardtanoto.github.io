Intermediate Algorithm Scripting: Missing letters





You will need to convert from character to ASCII code using fromCharCode() and charCodeAt().

Each alphabets have their own unicode, for example A is 65, and in binary 65 is 01000001, which is A

To find a missing letter like in ABCE, the unicode will be something like 65 66 67 69, so the missing unicode is 68, which is D

Here is the solution:



function fearNotLetter(str) {
for(var i = 0; i < str.length; i++) {
  /* code of current character */
  var code = str.charCodeAt(i);

     /* if code of current character is not equal to first character + no of iteration
     hence character has been escaped */
     if (code !== str.charCodeAt(0) + i) {

     /* if current character has escaped one character find previous char and return */
     return String.fromCharCode(code - 1);
    }
   }
  return undefined;
}

Moving zeros to the end 5 kyu codewars challenge


Write an algorithm that takes an array and moves all of the zeros to the end, preserving the order of the other elements.

moveZeros([false,1,0,1,2,0,1,3,"a"]) // returns[false,1,1,2,1,3,"a",0,0]

Read this article on declarative and imperative programming
There are 2 ways to solve this challenge, an imperative way or declarative way, I'll start with the imperative way.



First, notice the result. returns[false,1,1,2,1,3,"a",0,0]
We can separate two arrays, for example array that has zero and array with no zero, then we add the 2 arrays together.


We will iterate through the array to check if the value is zero or not, if its zero, we want to store it in array with other zeroes, and if it isnt zero, we want to store it in array with no zeroes, then we can just unite the 2 array together.


We add the array with concat method

Here is the imperative solution:

var moveZeros = function (arr) {
var result = [];
var zeroes = [];
   for(var i = 0;i
     if(arr[i] !== 0){
      result.push(arr[i])
     }
     else{
      zeros.push(arr[i])
     }
   }
   return result.concat(zeros)
}


In the declarative solution, you write code that describes what you want, but not necessarily how to get it (declare your desired results, but not the step-by-step), what we want here is to put an array that has zeroes after array that has no zeroes.

Here is the declarative solution:

var moveZeros = function (arr) {
     return arr.filter(x=>x !== 0).concat(arr.filter(x=>x===0 ))
}

What this code does is simply find the element inside array, which is x, that isnt zero, and concat it to another array which is an array with zeroes.

Regular Expression

Regular expressions are special strings that represent a search pattern. Also known as "regex" or "regexp", they help programmers match, search, and replace text

Regex starts with / and end with / too.

Example (Source from freeCodeCamp)

Return true if the given string is a palindrome. Otherwise, return false.
A palindrome is a word or sentence that's spelled the same way both forward and backward, ignoring punctuation, case, and spacing.
Note

You'll need to remove all non-alphanumeric characters (punctuation, spaces and symbols) and turn everything into the same case (lower or upper case) in order to check for palindromes.
We'll pass strings with varying formats, such as "racecar", "RaceCar", and "race CAR" among others.
We'll also pass strings with special symbols, such as "2A3*3a2", "2A3 3a2", and "2_A3*3#A2".

Solution:


function palindrome(str) {
   return str.replace(/[\W_]/g, '').toLowerCase() ===
   str.replace(/[\W_]/g, '').toLowerCase().split('').reverse().join(''); }



Imagine writing 2 line of code that check if race cAr(ignore case) is palindrome or not, regex is worth to master.

What is str.replace(/[\W_]/g, '').toLowerCase()

First, we want the argument (str) to replace everything that matches with /[\W_]/g with ''.

/[\W_]/g

\W Matches any character that is not a word character (alphanumeric & underscore)

_ matches with "_" character

g after the regex is a flag, there are many types of flags, and g is a flag for global which find all matches rather than stopping after the first match

Now for str.replace(/[\W_]/g, '').toLowerCase().split('').reverse().join('');, its the same thing.

The code will ignore the cases by using toLowerCase(), and then it will return true if its ===.
