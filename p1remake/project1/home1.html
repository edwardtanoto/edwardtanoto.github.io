Multiplayer Offline Game




Project 1

    This is my first project:

  • The game has 4 areas
  • The concept of the game is boardgame
  • This game is a mindgame and strategic game
  • This game requires 2 people offline
  • You can be creative with your own area
  • Defeat the opponents by using skills
  • Skills are added to make the game interesting
  • This simple game is created to have fun, so have some fun!!

    How to Play

  • There are 8 characters
  • 4 warriors is the maximum for each area
  • Remember you only can use 10 warriors in 4 areas
  • Some characters have their own skills, sniper and assassin
  • There are some combos between some characters
  • Area win is decided by the resultant of player 1 and player 2 scores
  • The winner is the one with most area won
  • You can type more than 4 letters to confuse your opponent
  • Remember to be fair and have fun
